== Description ==

Phu Concepts Inc’s admin theme. Once activated, all new user registrations will default to the Phu color scheme.

== Installation ==

1. Download through plugin directory or manually upload
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Users >> edit user >> Admin Color Scheme >> Phu
